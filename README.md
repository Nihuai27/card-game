# 🎮 卡牌冒险游戏

一个DND风格的卡牌战斗游戏，集成了AI生成的随机事件和战前对话。

## 🚀 在线试玩

[点击这里开始游戏](你的Vercel链接)

## ✨ 游戏特色

- 🎭 12种不同职业的英雄
- 👹 多种怪物和Boss
- 🤖 AI生成的战前对话
- 🎲 随机事件系统
- ⚔️ 卡牌战斗机制

## 🛠️ 本地开发

```bash
# 克隆项目
git clone <你的仓库地址>

# 进入目录
cd 卡牌

# 本地预览
npx serve .
```

## 📦 部署到 Vercel

```bash
# 安装 Vercel CLI
npm i -g vercel

# 登录
vercel login

# 部署
vercel --prod
```

## 📝 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 阿里通义千问 API
